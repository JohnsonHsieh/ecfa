作者vanilla1365 (小瓜呆)看板FuMouDiscuss標題Re: [建議] 我們可以作的下一步，請幫轉給核心成員時間Tue Mar 25 11:34:13 2014
※ 引述《hickory (C'est la vie)》之銘言：
: 本人是彰化縣國中老師正職  對教育與大眾傳播有點專業
: 在此提出一些看法，請所有能仁志士一同討論，
: 也請將此文轉給在議場中辛苦堅持的核心成員知道。
: 現在全台灣可以常常上網的大概都已經知道服貿事情鬧很大
: 而這群可以常常上網(尤其是有在用臉書)的人，其實多多少少都對服貿有一些了解
: 他們並不是我們要說服的對象
: 就我自己在全台灣第二大的社區大學上課，觀察發現大多數民眾的現況是
: 知道有學生在鬧服貿，不太清楚她們在鬧什麼。
: 但是他們非常想知道服貿是什麼，為什麼要反，卻沒有管道知道。
: 也就是說我們缺少 "說服的力量"
: 今天看了蘋果，照片很溫暖，有QR Code。
: OK，你覺得40歲以下，願意拿出手機平板掃一掃，讀文章的人有多少？
: 那你覺得40歲以上，有手機平板，會去掃QR Code 的人又有多少？
: 就教育角度，我會認為這種教學方法的效果是很低的。
: 別誤會，我並不反對 "買頭版" 這個作法，
: 但我認為要讓更多的 "中間選民" 了解服貿，進而知道為何而反，需要其他更有效方式
: 就教育的角度來看，假設把民眾比喻成是未教育的。
: 買頭版廣告放QR Code這件事情，有點像是把一本講義丟給一個班，叫他們自己去看
: 會去看的大概只有前段那些有能力有意願的學生 中段跟後段 其實是被放棄的
: 然而他們這一群，卻是我們這一仗能不能打贏的關鍵所在。
: 我們是否能夠想個方式，讓這些中段或後段，在資訊能力較弱的民眾也能了解服貿，
: 進而加入我們，讓征戰長路之後的立委罷免、公投等可行的策略好走一些？
: 我想了一下，我認為有些方式可行
: 1.報紙：買下一整週的頭版，針對服貿的法制、民生、影響、醫療、國安等層面
:   以詳細說明的方式詳列上去，作成連載。
:   本作法的好處是一但連載開始，
:   該報後續幾天的發行量會大增，有助於價錢的談判籌碼。
: 2.報紙：不作連載，改作八個版的專刊。各版寫各層面開放什麼，有什麼影響，
:   好處或壞處都詳細說明。專刊的好處是自由、蘋果等媒體願意配合的機會高，
:   其他黨軍報紙不見得願意配合。但這有解套方法，就是夾報。
:   買通報紙物流，用夾帶的方式一樣可以把訊息送到四大報的主要讀者眼前。
: 3.電視：請反服貿陣營推派人上電視辯論。
:   黃國昌老師、賴中強律師在制度層面上的專長。
:   醫生上去講醫療照護商品化，
:   有國安專長的上去分析開放後國安方面的影響
:   不過這方法牽涉到電視台是否願意的問題  難度較高
: 至於方法1跟方法2的所需經費 可以跟之前一樣網路募款
: 順帶一提，買頭版那天早上九點開放，本來想捐兩萬，
: 誰知短短半小時  我上個課回來已經截止  假如開放8小時，搞不好能捐到幾千萬。
: 很多 "大人" 都說，
: 我們這些大人沒用，怕事。為你們捐款是我們唯一能作的。
: 我們能不能贏，要看我們有多想贏！
:                                      -出自公民1985演講稿

個人認為這想法太重要了，實在不希望被忽視
在現場的朋友基本上理念一致，但是口號或短講的內容千奇百怪，也很容易分散焦點
一下退回服貿、一下逐條審查、還有一邊一國、台灣獨立的

我想補充的是
1.面對志同道合的朋友需要灌注多一點民氣、例如回憶324鎮壓事件
  讓大家願意持續到國會現場、同時防止抗爭流於園遊會型式。
2.面對不支持或容易受媒體影響的朋友，真的需要更多說服的力量，
  要針對政府似是而非的言論講解、辯論，才能凸顯他們的愚民政策啊。
3.另外最近幾天國會現場有很多人再發傳單，可惜發錯地點了，
  傳單應該要發給立場相左的人看，去火車站或其他捷運站可能比較有效果

不要放棄說服冷漠的人，籌碼才會增加，勝利才會屬於我們
