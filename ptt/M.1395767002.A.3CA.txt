: 他們集中要抓一個同學
: 就因為要奮力拉他回來,在場得人都遭到毆打

: 大家無能為力
: 只能撤回辦公室
: 這辦公室就是電視上說被破壞的那個
: 我們進去並沒有破壞任何東西
: 只有把鐵櫃搬去檔門
: 電視上說玻璃被打破
: 報紙寫鐵櫃被打爛還附上圖片
: 這些都是警察攻堅的時候打破,打壞的
: 並不是由學生破壞
: 更扯的是說學生吃太陽餅,蛋糕,喝黑麥汁
: 大家想想看
: 學生防守都沒來不及了
: 還有時間吃喝嗎?
: 會去相信媒體的人
: 我也不知道該講甚麼了
: 後來辦公室守不住
: 大家就排隊下樓梯撤回廣場
: 排在隊伍最後的幾個撤得太慢就被警察抓走
: 某A 就是其中一個
: 不久後我加入靜坐行列
: 坐在行政院門口第二排
: 到了2點左右
: 鎮暴警察就開始行動了
: 分別從北平東路跟天津街
: 當時聽到廣播說"在那裡的同學已經被警察攻擊了"
: 就因為那邊沒有媒體
: 警察下手就特別重
: 我聽了非常難過
: 不敢相信自己所聽到的事實
: 到了4點多
: 警察就已經攻到行政院門口的兩側
: 當時看到左邊的同學先被水車攻擊
: 再被警察用警棍,圓盾,長盾打
: 還有警察追著學生打
: 真的是殺紅了眼
: 眼見陸續有傷患被抬出場
: 流血的流血
: 哀嚎的哀嚎
: 我真不敢相信自己所看到的一切
: 眼淚都掉了出來
: 過不了多久
: 只剩門口的我們
: 當時被警察包圍
: 媒體都被趕出場了
: 先是水車噴我們
: 在一個一個抓
: 有個學生因為躺在地上掙扎不給他們抬
: 就被警察們當場圍毆
: 有盾牌的用盾牌
: 沒盾牌的用腳
: 那位勇敢的學生被打嘴裡還不停喊著"捍衛民主"
: 看到這畫面
: 我已經淚流不止
: 心非常的痛
: 自己又無能為力
: 一有動作就是現行犯
: 只能看著他被打
: 心裡不停的祈禱這一切趕快停止
: 他也是別人的小孩
: 你們怎麼可以把他打成這樣
: 難到別人家的小孩就死不完嗎?
: 後來他的聲音就從口號轉為哀嚎
: 抱著左腳不停的顫抖
: 學生們不停的呼喊"醫生"
: 才有醫療人員過來把他抬走
: 當時他已經無法走路
: 並且發出淒厲的哀嚎聲
: 我真心覺得他腳應該被打斷了
: 接著就輪到我們了
: 眼見一個一個同學被拖走
: 掙扎就被毆打
: 當時的我已經無法控制自己的淚水
: 我前方的勇士某B.
: 就因為在被扛的時候抱住警察
: 當場被踹了好幾腳,打了好幾下
: 輪到我的時候
: 我只能說我太懦弱了
: 看到大家被打成這樣
: 我已經沒有勇氣反抗
: 被扛出場的過程沒有掙扎,放軟四支
: 所以只被踹了幾腳
: 出來後

: 傷心中又參雜著憤怒
: 我不斷的告訴自己要冷靜

: 就去幫他開路

: 這時我又忍不住流下淚水
: 心裡非常得愧疚
: 沒能保護好自己得朋友
