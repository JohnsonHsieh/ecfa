: 臺灣大學學生會
: 聲明：警政採暴力方式攻擊學生　臺大學生會即日起呼籲自主罷課
: 　　
: 　　為維護民主法治憲政及程序正義，對「海峽兩岸服務貿易協議」之黑箱審議及通過過
: 程表達不滿，學生自主佔領立法院進行和平抗爭活動已逾6日。然政府當局除至今仍未正
: 面回應群眾訴求外，更罔顧群眾人身安全，於昨（23）日採取暴力方式進行鎮壓。為此，
: 國立臺灣大學學生會（下稱臺大學生會）特發此聲明譴責有關當局之過當手段，並將呼籲
: 各大專校院校方及其學生自治組織進行自主罷課。
: 　
: 　　臺大學生會認為，面對本次抗爭活動，參與學生及群眾先前已透過和平手段展現了公
: 民水準，並且對於訴求目的十分明確且理性。然而，政府當局非但沒有正面回應群眾訴求
: ，反而採取過當手段以武力攻擊學生、進行暴力鎮壓，傷害國家實質的主人，實嚴重踐踏
: 了基本人權；臺大學生會嚴厲譴責警政暴力，並要求政府當局負起責任。
: 　
: 　　臺大學生會基於保護學生人身安全的責任感與目的，除了先前（19）日已發佈聲明呼
: 籲有關當局切勿動用暴力手段外，也將儘快與臺大校方進行溝通。期望校方尊重學生會及
: 師生參與公共事務外，並與師生站在同一陣線，保護師生的安全。
: 　
: 　　此外，針對外界質疑學校不應任意停課，侵犯學生受教權，臺大學生會認為師生溝通
: 調整課程屬講學自由，於立法院外實踐公民運動更同屬學習的一環；故學生應積極介入，
: 促成社會的改變。臺大學生會於是呼籲各大專院校學生會，共同呼應由學生自主發起的罷
: 課聯署，亦提供下列幾點做法，期望師生共同溝通協商：
: 　
: （1）系所主任、老師直接宣布停課停教，學生全面上街。
: （2）學生表達上街學習的立場，向老師提出集體「換教室」的請求。
: （3）學生片面表達上街學習的立場，要求老師不對上街的學生進行處分，老師們也繼續
: 維持在學校內的教學工作，不上街頭的同學也可保有體制內的受教權。
: （4）老師片面宣布不對上街頭同學進行處置，支持學生上街頭學習，並維持教學工作（
: 或更改教學內容，配合議題）。　　
: 　
: 　　期望在面對臺灣民主法治憲政遭踐踏的此刻，學生族群能夠正視問題並挺身而出，在
: 注意自身安全及相關法律責任的前提下以行動為民主進程盡一份力，直至政府當局公開道
: 歉並正面回應學生訴求，讓臺灣的民主最終能夠再一次的進步。
